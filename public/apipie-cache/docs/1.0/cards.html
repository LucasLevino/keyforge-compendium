Cards
The cards API lets you lookup all individual cards.

>>>

GET /api/v1/cards/:id.json
Show card details

GET /api/v1/sets/:set_id/cards/:id.json
Show card details for given set

Params

Param name Description
id
optional

Card number

Validations:

  • Must be a Integer
set_id
optional

Set id. 1 for “Call of the Archons”, 2 for “Age of Ascension”

Validations:

  • Must be one of: 1, 2.

Returns

Code: 200

Description:

OK
