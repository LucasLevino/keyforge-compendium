• Keyforge Compendium API 1.0

The Keyforge Compendium API uses 'Basic Access Authentication' where the key is the username and the secret is the password. Most libraries come with build in support for this if not you will have to built the header yourself.

Resources

Cards
The cards API lets you lookup all individual cards.

Resource Description
GET /api/v1/cards/:id.json Show card details
GET /api/v1/sets/:set_id/cards/:id.json Show card details for given set

Decks
The decks API lets you lookup individual decks and their various properties such as ADHD and SAS ratings.

Resource Description
GET /api/v1/decks/:id Show deck details
GET /api/v1/decks List all decks
GET /api/v1/decks/by_name/:name Search on decks by name
GET /api/v1/decks/random/:amount Grab a few random deck IDs
